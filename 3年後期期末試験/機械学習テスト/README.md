# 機械学習 中間テスト対策 一問一答アプリ

機械学習の中間試験過去問を美しく収録し、学習効率を高める機能を備えた一問一答型のWebアプリケーション（SPA）です。

## 🚀 特徴
- **過去問準拠の完璧な設問**: 『機械学習過去問問題と解答まとめ.pdf』に記載された設問、選択肢、数式表現、および解説を完全に再現。
- **統計分析の表組み対応**: 問2の複雑な統計データ成績表を等幅テキストで綺麗にフォーマットして埋め込み。
- **選択式・記述式のデュアル学習モード**: 問4（過学習・交差検証）では、本番通りの4択テスト形式と、重要なキーワードの含有量を判定する記述式練習モードの両方を搭載。
- **学習サポート機能**: 
  - 後で復習したい問題の「ブックマーク（お気に入り）」機能
  - 間違えた問題だけをまとめて解き直す「要復習モード」
  - 解答履歴とお気に入り状態を `localStorage` でローカル保存。ブラウザを閉じても進捗が失われません。
- **近未来SF調デザイン**: 目に優しいダークテーマ、洗練されたネオンカラーアクセント、グラスモルフィズムを取り入れたUI。

## 🛠️ 技術スタック
- **構造**: HTML5 (セマンティックマークアップ)
- **スタイル**: CSS3 (Vanilla CSS, フレキシブルグリッド, レスポンシブ対応)
- **ロジック**: Vanilla JavaScript (ES6+, 状態管理, localStorage)

## 📦 ローカルでの起動方法
特別なサーバー設定やビルド手順は不要です。
1. リポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザ（Chrome, Safari, Edgeなど）で開きます。

## ☁️ Vercelでの公開手順
Vercelは静的サイトを無料でホストできます。
1. このリポジトリをご自身のGitHubアカウントにプッシュ（プライベート推奨）します。
2. [Vercel](https://vercel.com/) にログインし、**「Add New Project」** をクリックします。
3. クローンした本リポジトリを選択してインポートし、そのまま **「Deploy」** ボタンを押します（数秒で公開されます）。
