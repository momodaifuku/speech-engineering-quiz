📋 重要用語チートシート

期末試験に出る重要概念・用語を一気にチェックしよう

消費者余剰
消費者の支払意思額（WTP）と実際の価格との差。
消費者余剰 = WTP − 価格
生産者余剰
実際の価格と機会費用（生産に必要な最低価格）との差。
生産者余剰 = 価格 − 機会費用
総余剰（社会的余剰）
総余剰 = 消費者余剰 + 生産者余剰
完全競争市場で最大化される
完全競争市場
条件: ①多数の買い手・売り手（価格受容者）②情報の対称性 ③取引費用ほぼゼロ ④同質財
上級財 / 下級財
所得が増えたとき需要が増える財 → 上級財
所得が増えたとき需要が減る財 → 下級財
需要の価格弾力性
価格1%変化 → 需要量が1%以上変化 → 弾力的
価格1%変化 → 需要量が1%未満変化 → 非弾力的
※生活必需品は非弾力的
代替財 / 補完財
代替財の価格↑ → 注目財の需要右シフト↑
補完財の価格↑ → 注目財の需要左シフト↓
機会費用
会計上の費用 + 見えない費用（あきらめた最善の選択肢から得られた利益）
価格の上限規制（天井価格）
均衡価格より低い上限設定 → 超過需要発生 → 死荷重・品質低下
最低賃金（価格下限規制）
均衡賃金より高い下限設定 → 超過供給（失業）発生 → 死荷重
死荷重（デッドウェイトロス）
価格規制・税・独占などによって失われる社会的余剰の純損失。完全競争時との差
費用便益分析
政策評価の手法。便益（B） / 費用（C） > 1 なら実施の目安
取引費用
契約・交渉・移動などに伴う費用。大きすぎると本来成立するはずの取引が妨げられ、総余剰が減少
効率性 vs 公平性
政府の所得再分配などで公平性↑ → 効率性（社会的余剰）↓ というトレードオフ関係
独占企業の特徴
・価格設定者（price setter）
・市場全体の需要曲線を知っている
・供給曲線が存在しない
利潤最大化条件
MR = MC（限界収入 = 限界費用）
これを満たす生産量Qを選択
独占の問題点
社会的に望ましい量より過少生産・高価格設定 → 死荷重の発生
法による独占
政府が与えた排他的権利。例：特許・著作権（イノベーション促進の目的）
自然独占
インフラ投資が大きく、1社独占が効率的な状態。例：ガス・電気・水道事業
カルテル vs 談合
カルテル：同業他社間での価格・生産量の共同決定
談合：入札前に落札者と金額を事前に協定
情報の非対称性
取引の一方が他方より多くの情報を持っている状態。→ 市場の失敗を引き起こす
逆選択（逆淘汰）
契約前の情報の非対称性。質の悪いものだけが市場に残る。
例：メルカリで粗悪品だけが出回る
モラルハザード
契約後の情報の非対称性。相手が見ていないとサボる。
例：保険加入後に不注意になる
シグナリング
情報を持つ側が自分の質の良さを相手に伝える行為
例：資格・学歴・評価の公開
スクリーニング
情報を持たない側が複数のメニューを提示し、相手のタイプを見分ける行為
例：保険の免責金額の差
モニタリング
プリンシパルがエージェントの行動を監視・監査することでモラルハザードを防ぐ
プリンシパル / エージェント
プリンシパル（本人）：仕事を依頼する側
エージェント（代理人）：仕事を請け負う側
外部性
市場を通さず第三者に直接影響を与えること。
負の外部性→過大生産、正の外部性→過少生産
負の外部性の是正
ピグー税（汚染に課税）または規制（内部化）
社会的費用 = 私的費用 + 外部費用
ピグー補助金
正の外部性がある行動を促進するために政府が与える補助金
例：ハイブリッド車の導入補助
純粋公共財
非排除性（利用制限不可） + 非競合性（消費が他に影響しない）
例：国防・道路・灯台
フリーライダー問題
非排除的な財において、対価を払わずに消費するタダ乗り問題。民間供給が成立しない理由
共有資源（コモンズ）
非排除性 + 競合性を持つ財。例：漁業資源・牧草地
→ 共有地の悲劇（過剰使用で枯渇）
財の4分類
排除性あり排除性なし
競合性あり私的財共有資源
競合性なしクラブ財公共財
労働市場の構造
労働需要曲線：企業の行動（右下がり）
労働供給曲線：労働者の行動（右上がり）
最低賃金の効果
均衡賃金より高い場合：
・需要量 Q1（企業の雇用）↓
・供給量 Q2（働きたい人）↑
→ 超過供給（失業）＋死荷重
労働生産性
労働者1人当たりが生み出す付加価値。労働生産性↑ → 賃金↑の傾向
名目賃金 vs 実質賃金
名目賃金：物価調整前の賃金（日本では横ばい）
実質賃金：物価調整後の購買力を示す賃金
効率賃金仮説
相場より高い賃金を払うことで、労働者のモラルハザードを防ぎ生産性を高めるという仮説
春闘
日本の労使の団体交渉（春の賃金交渉）の仕組み。毎年3〜4月に実施
裁量労働制
労働者が出退勤・仕事の進め方を決める働き方。企業は残業代の支払い義務なし。懸念は長時間労働
労働分配率
付加価値のうち労働者が賃金として受け取った割合。日本では低下傾向（内部留保の蓄積）
